## 光标

Tailwind v4 [切换了](https://tailwindcss.zhcndoc.com/docs/upgrade-guide#buttons-use-the-default-cursor)按钮组件的光标样式，从 `cursor: pointer` 改为 `cursor: default`。

如果你想保留 `cursor: pointer` 行为，请在你的 CSS 文件中添加以下代码：

```css showLineNumbers title="globals.css"
@layer base {
  button:not(:disabled),
  [role="button"]:not(:disabled) {
    cursor: pointer;
  }
}
```

### 链接

你可以使用 `asChild` 属性让其他组件看起来像按钮。下面是一个看起来像按钮的链接示例。

```tsx showLineNumbers
import Link from "next/link"

import { Button } from "@/components/ui/button"

export function LinkAsButton() {
  return (
    <Button asChild>
      <Link href="/login">登录</Link>
    </Button>
  )
}
```

## API 参考

### Button

`Button` 组件是对 `button` 元素的封装，添加了多种样式和功能。

| 属性      | 类型                                                                                  | 默认值      |
| --------- | ------------------------------------------------------------------------------------- | ----------- |
| `variant` | `"default" \| "outline" \| "ghost" \| "destructive" \| "secondary" \| "link"`         | `"default"` |
| `size`    | `"default" \| "sm" \| "lg" \| "icon" \| "icon-sm" \| "icon-lg"`                       | `"default"` |
| `asChild` | `boolean`                                                                            | `false`     |

## 更新日志

### 2025-09-24 新增尺寸

我们为按钮组件新增了两个尺寸：`icon-sm` 和 `icon-lg`，用于创建图标按钮。要添加它们，请编辑 `button.tsx`，在 `buttonVariants` 中的 `size` 下添加以下代码：

```tsx showLineNumbers title="components/ui/button.tsx"
const buttonVariants = cva("...", {
  variants: {
    size: {
      // ...
      "icon-sm": "size-8",
      "icon-lg": "size-10",
      // ...
    },
  },
})
```
