## 使用方法

```tsx showLineNumbers
import {
  HoverCard,
  HoverCardContent,
  HoverCardTrigger,
} from "@/components/ui/hover-card"
```

```tsx showLineNumbers
<HoverCard>
  <HoverCardTrigger>悬停</HoverCardTrigger>
  <HoverCardContent>
    React 框架 — 由 @vercel 创建和维护。
  </HoverCardContent>
</HoverCard>
```

## 触发延迟

在触发组件上使用 `delay` 和 `closeDelay` 来控制卡片何时打开和关闭。

```tsx showLineNumbers
<HoverCard>
  <HoverCardTrigger delay={100} closeDelay={200}>
    悬停
  </HoverCardTrigger>
  <HoverCardContent>内容</HoverCardContent>
</HoverCard>
```

## 位置设置

使用 `HoverCardContent` 上的 `side` 和 `align` 属性控制位置。

```tsx showLineNumbers
<HoverCard>
  <HoverCardTrigger>悬停</HoverCardTrigger>
  <HoverCardContent side="top" align="start">
    内容
  </HoverCardContent>
</HoverCard>
```
