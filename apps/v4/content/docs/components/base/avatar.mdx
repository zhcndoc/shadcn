## API 参考

### Avatar

`Avatar` 组件是包装头像图像和回退内容的根组件。

| 属性        | 类型                        | 默认值      |
| ----------- | --------------------------- | ----------- |
| `size`      | `"default" \| "sm" \| "lg"` | `"default"` |
| `className` | `string`                    | -           |

### AvatarImage

`AvatarImage` 组件用于显示头像图像。它接受所有 Base UI 头像图像属性。

| 属性        | 类型     | 默认值 |
| ----------- | -------- | ------- |
| `src`       | `string` | -       |
| `alt`       | `string` | -       |
| `className` | `string` | -       |

### AvatarFallback

`AvatarFallback` 组件在图像加载失败时显示回退内容。它接受所有 Base UI 头像回退属性。

| 属性        | 类型     | 默认值 |
| ----------- | -------- | ------- |
| `className` | `string` | -       |

### AvatarBadge

`AvatarBadge` 组件在头像上显示徽章指示器，通常位于右下角。

| 属性        | 类型     | 默认值 |
| ----------- | -------- | ------- |
| `className` | `string` | -       |

### AvatarGroup

`AvatarGroup` 组件显示带有重叠样式的头像组。

| 属性        | 类型     | 默认值 |
| ----------- | -------- | ------- |
| `className` | `string` | -       |

### AvatarGroupCount

`AvatarGroupCount` 组件在头像组中显示计数指示器，通常表示额外头像数量。

| 属性        | 类型     | 默认值 |
| ----------- | -------- | ------- |
| `className` | `string` | -       |

更多有关 Base UI 头像属性的信息，请参阅 [Base UI 文档](https://base-ui.com/react/components/avatar#api-reference)。
