## API 参考

### AspectRatio

`AspectRatio` 组件用于以期望的比例展示内容。

| 属性         | 类型       | 默认值 | 必填   |
| ----------- | ---------- | ------ | ------ |
| `ratio`     | `number`   | -      | 是     |
| `className` | `string`   | -      | 否     |

更多信息请参见 [Base UI 文档](https://base-ui.com/react/components/aspect-ratio#api-reference)。
