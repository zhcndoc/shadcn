## API 参考

### AspectRatio

`AspectRatio` 组件用于在指定的宽高比范围内显示内容。

| 属性         | 类型       | 默认   | 是否必填 |
| ------------ | ---------- | ------ | -------- |
| `ratio`      | `number`   | -      | 是       |
| `className`  | `string`   | -      | 否       |

更多信息，请参见 [Radix UI 文档](https://www.radix-ui.com/primitives/docs/components/aspect-ratio#api-reference)。
