## API 参考

### Kbd

使用 `Kbd` 组件显示键盘按键。

| 属性        | 类型     | 默认值  |
| ----------- | -------- | ------- |
| `className` | `string` | ``      |

```tsx
<Kbd>Ctrl</Kbd>
```

### KbdGroup

使用 `KbdGroup` 组件将多个 `Kbd` 组件分组。

| 属性        | 类型     | 默认值  |
| ----------- | -------- | ------- |
| `className` | `string` | ``      |

```tsx
<KbdGroup>
  <Kbd>Ctrl</Kbd>
  <Kbd>B</Kbd>
</KbdGroup>
```
