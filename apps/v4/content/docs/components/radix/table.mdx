## 使用方法

```tsx showLineNumbers
import {
  Table,
  TableBody,
  TableCaption,
  TableCell,
  TableHead,
  TableHeader,
  TableRow,
} from "@/components/ui/table"
```

```tsx showLineNumbers
<Table>
  <TableCaption>你最近发票的列表。</TableCaption>
  <TableHeader>
    <TableRow>
      <TableHead className="w-[100px]">发票</TableHead>
      <TableHead>状态</TableHead>
      <TableHead>方式</TableHead>
      <TableHead className="text-right">金额</TableHead>
    </TableRow>
  </TableHeader>
  <TableBody>
    <TableRow>
      <TableCell className="font-medium">INV001</TableCell>

      <TableCell>信用卡</TableCell>
      <TableCell className="text-right">$250.00</TableCell>

  </TableBody>
</Table>
```

## 数据表格

你可以使用 `<Table />` 组件构建更复杂的数据表格。结合 [@tanstack/react-table](https://tanstack.com/table/v8) 来创建带有排序、筛选和分页功能的表格。

更多信息请参见 [数据表格](/docs/components/data-table) 文档。

你也可以在 [任务](/examples/tasks) 示例中查看数据表格的示例。
