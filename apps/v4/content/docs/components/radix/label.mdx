```tsx

```

```tsx
<Label htmlFor="email">您的邮箱地址</Label>
```

## Field 中的标签

对于表单字段，请使用包含内置 `FieldLabel`、`FieldDescription` 和 `FieldError` 组件的 [Field](/docs/components/radix/field) 组件。

```tsx
<Field>
  <FieldLabel htmlFor="email">您的邮箱地址</FieldLabel>
  <Input id="email" />
</Field>
```
