## 用法

```tsx showLineNumbers
import { Label } from "@/components/ui/label"
import { RadioGroup, RadioGroupItem } from "@/components/ui/radio-group"
```

```tsx showLineNumbers
<RadioGroup defaultValue="option-one">
  <div className="flex items-center gap-3">
    <RadioGroupItem value="option-one" id="option-one" />
    <Label htmlFor="option-one">选项一</Label>
  </div>
  <div className="flex items-center gap-3">
    <RadioGroupItem value="option-two" id="option-two" />
    <Label htmlFor="option-two">选项二</Label>
  </div>
</RadioGroup>
```

## 示例

### 描述

使用 `Field` 组件添加描述的单选组项。

<ComponentPreview styleName="radix-nova" name="radio-group-description" />

### 选择卡片

使用 `FieldLabel` 包裹整个 `Field`，实现可点击的卡片式选择。

<ComponentPreview styleName="radix-nova" name="radio-group-choice-card" />

### 字段集

使用 `FieldSet` 和 `FieldLegend` 对单选项进行分组，并添加标签和描述。

<ComponentPreview styleName="radix-nova" name="radio-group-fieldset" />

### 禁用

在 `RadioGroupItem` 上使用 `disabled` 属性禁用单个选项。

<ComponentPreview styleName="radix-nova" name="radio-group-disabled" />

### 无效

在 `RadioGroupItem` 上使用 `aria-invalid`，在 `Field` 上使用 `data-invalid` 来显示验证错误。

<ComponentPreview styleName="radix-nova" name="radio-group-invalid" />
