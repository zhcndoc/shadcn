## 示例

### 变体

使用 `variant` 属性更改徽章的变体。

<ComponentPreview styleName="radix-nova" name="badge-variants" />

### 带图标

您可以在徽章内部渲染图标。使用 `data-icon="inline-start"` 将图标渲染在左侧，使用 `data-icon="inline-end"` 将图标渲染在右侧。

<ComponentPreview styleName="radix-nova" name="badge-icon" />

### 带加载动画

您可以在徽章内部渲染加载动画。请记得为加载动画添加 `data-icon="inline-start"` 或 `data-icon="inline-end"` 属性。

<ComponentPreview styleName="radix-nova" name="badge-spinner" />

### 链接

使用 `asChild` 属性将链接渲染为徽章。

<ComponentPreview styleName="radix-nova" name="badge-link" />

### 自定义颜色

您可以通过添加自定义类（如 `bg-green-50 dark:bg-green-800`）到 `Badge` 组件来定制徽章的颜色。

<ComponentPreview styleName="radix-nova" name="badge-colors" />

## API 参考

### Badge

`Badge` 组件用于显示徽章或看起来像徽章的组件。

| 属性        | 类型                                                                             | 默认值       |
| ----------- | -------------------------------------------------------------------------------- | ------------ |
| `variant`   | `"default" \| "secondary" \| "destructive" \| "outline" \| "ghost" \| "link"`    | `"default"`  |
| `className` | `string`                                                                         | -            |
