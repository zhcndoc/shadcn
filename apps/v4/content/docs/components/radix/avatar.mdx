## API 参考

### Avatar

`Avatar` 组件是根组件，用于包裹头像图片和回退内容。

| 属性        | 类型                        | 默认值     |
| ----------- | --------------------------- | --------- |
| `size`      | `"default" \| "sm" \| "lg"` | `"default"` |
| `className` | `string`                    | -         |

### AvatarImage

`AvatarImage` 组件展示头像图片。它接受所有 Radix UI Avatar Image 属性。

| 属性        | 类型     | 默认值 |
| ----------- | -------- | ------ |
| `src`       | `string` | -      |
| `alt`       | `string` | -      |
| `className` | `string` | -      |

### AvatarFallback

`AvatarFallback` 组件在图片加载失败时展示回退内容。它接受所有 Radix UI Avatar Fallback 属性。

| 属性        | 类型     | 默认值 |
| ----------- | -------- | ------ |
| `className` | `string` | -      |

### AvatarBadge

`AvatarBadge` 组件在头像上显示徽章指示器，通常位于右下角。

| 属性        | 类型     | 默认值 |
| ----------- | -------- | ------ |
| `className` | `string` | -      |

### AvatarGroup

`AvatarGroup` 组件展示一组头像，带有重叠样式。

| 属性        | 类型     | 默认值 |
| ----------- | -------- | ------ |
| `className` | `string` | -      |

### AvatarGroupCount

`AvatarGroupCount` 组件在头像组中显示数量指示器，通常表示额外头像的数量。

| 属性        | 类型     | 默认值 |
| ----------- | -------- | ------ |
| `className` | `string` | -      |

更多关于 Radix UI Avatar 属性的信息，请参阅 [Radix UI 文档](https://www.radix-ui.com/primitives/docs/components/avatar#api-reference)。
