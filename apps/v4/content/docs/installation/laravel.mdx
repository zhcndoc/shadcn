title: Laravel
description: 为 Laravel 安装和配置 shadcn/ui
---

<Steps>

### 创建项目

首先使用 Laravel 安装器创建一个带有 Inertia 和 React 的新 Laravel 项目：

```bash
laravel new my-app --react
```

### 添加组件

现在你可以开始向项目中添加组件。

```bash
npx shadcn@latest add switch
```

上述命令会将 `Switch` 组件添加到 `resources/js/components/ui/switch.tsx`。然后你可以这样导入它：

```tsx title="resources/js/pages/index.tsx" {1,6} showLineNumbers
import { Switch } from "@/components/ui/switch"

const MyPage = () => {
  return (
    <div>
      <Switch />
    </div>
  )
}

export default MyPage
```

</Steps>
