<Callout className="mb-6 border-emerald-600 bg-emerald-100 dark:border-emerald-400 dark:bg-emerald-900">

**开始一个新项目？** 使用 [shadcn/create](/create) 来快速获得一个配置完备的 Laravel 应用，包含自定义主题、Base UI 或 Radix 以及图标库。

</Callout>

<Steps>

### 创建项目

首先使用 Laravel 安装器创建一个带有 Inertia 和 React 的新 Laravel 项目：

```bash
laravel new my-app --react
```

### 运行 shadcn/ui CLI

运行 `init` 命令，在你的 Laravel 项目中配置 shadcn/ui：

```bash
npx shadcn@latest init --force
```

如果系统提示是否覆盖已有组件，输入 `y` 继续。

### 添加组件

现在你可以开始向项目中添加组件。

```bash
npx shadcn@latest add switch
```

上述命令会将 `Switch` 组件添加到 `resources/js/components/ui/switch.tsx`。然后你可以这样导入它：

```tsx title="resources/js/pages/index.tsx" {1,6} showLineNumbers
import { Switch } from "@/components/ui/switch"

const MyPage = () => {
  return (
    <div>
      <Switch />
    </div>
  )
}

export default MyPage
```

</Steps>
