title: TanStack Start
description: 安装并配置 shadcn/ui 以用于 TanStack Start。
---

<Steps>

### 创建项目

运行以下命令以创建一个包含 shadcn/ui 的 TanStack Start 新项目：

```bash
npm create @tanstack/start@latest --tailwind --add-ons shadcn
```

### 添加组件

现在你可以开始向项目添加组件了。

```bash
npx shadcn@latest add button
```

上面的命令会将 `Button` 组件添加到你的项目中。然后可以这样导入它：

```tsx title="app/routes/index.tsx" showLineNumbers {1,6}
import { Button } from "@/components/ui/button"

function App() {
  return (
    <div>
      <Button>点击我</Button>
    </div>
  )
}
```

</Steps>

如果你想添加所有 `shadcn/ui` 组件，可以运行以下命令：

```bash
npx shadcn@latest add --all
```
