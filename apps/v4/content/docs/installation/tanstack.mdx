<Callout className="mb-6 border-emerald-600 bg-emerald-100 dark:border-emerald-400 dark:bg-emerald-900">

**开始一个新项目？** 使用 [shadcn/create](/create) 来获得一个完全配置好的 TanStack Start 应用，内含自定义主题、Base UI 或 Radix 以及图标库。

</Callout>

<Steps>

### 创建项目

运行以下命令以创建一个包含 shadcn/ui 的 TanStack Start 新项目：

```bash
npx shadcn@latest init -t tanstack
```

**对于 monorepo 项目，使用 `--monorepo` 标志：**

```bash
npx shadcn@latest init -t tanstack --monorepo
```

### 添加组件

现在你可以开始向项目添加组件了。

```bash
npx shadcn@latest add button
```

上面的命令会将 `Button` 组件添加到你的项目中。然后可以这样导入它：

```tsx title="app/routes/index.tsx" showLineNumbers {1,6}
import { Button } from "@/components/ui/button"

function App() {
  return (
    <div>
      <Button>点击我</Button>
    </div>
  )
}
```

</Steps>
