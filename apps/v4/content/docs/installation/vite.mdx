---
title: Vite
description: 为 Vite 安装和配置 shadcn/ui。

<Callout className="mb-6 border-emerald-600 bg-emerald-100 dark:border-emerald-400 dark:bg-emerald-900">

**从头开始？** 使用 [shadcn/create](/create) 来配置一个带有自定义主题、Base UI 或 Radix 以及图标库的 Vite 项目。

</Callout>

<Steps>

### 创建项目

运行 `init` 命令来创建新的 Vite 项目或设置现有项目：

```bash
npx shadcn@latest init -t vite
```

**对于 monorepo 项目，使用 `--monorepo` 标志：**

```bash
npx shadcn@latest init -t vite --monorepo
```

### 添加组件

现在你可以开始向项目中添加组件了。

```bash
npx shadcn@latest add button
```

上述命令会将 `Button` 组件添加到你的项目中。然后你可以这样导入它：

```tsx {1,6} showLineNumbers title="src/App.tsx"
import { Button } from "@/components/ui/button"

export default function App() {
  return (
    <div>
      <Button>点击我</Button>
    </div>
  )
}
```

</Steps>
