---

shadcn CLI 现在支持本地文件。从本地 JSON 文件初始化项目并添加组件、主题、hooks、工具等。

```bash
# 从本地文件初始化项目
npx shadcn init ./template.json

# 从本地文件添加组件
npx shadcn add ./block.json
```

此功能带来了强大的新工作流程：

- **零配置** - 无需远程注册表
- **更快的开发** - 在发布前本地测试注册表条目
- **为代理和 MCP 增强工作流** - 本地生成和运行注册表条目
- **私有组件** - 保持专有组件的本地和私密。
