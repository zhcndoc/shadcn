date: 2025-02-19
---

我们发布了 Tailwind v4 和 React 19 的首次预览版，随时欢迎您试用。您可以从今天开始使用它。

更新内容：

- CLI 现在可以初始化 Tailwind v4 项目。
- 完整支持新的 @theme 指令和 @theme 内联选项。
- 所有组件均已升级以支持 Tailwind v4 和 React 19。
- 我们移除了 forwardRefs，并调整了类型。
- 每个基础组件现在都有 data-slot 属性用于样式定制。
- 修复并优化了组件的样式。
- 我们正在弃用 toast 组件，推荐使用 sonner。
- 按钮现在使用默认光标。
- 我们正在弃用默认样式，新项目将使用 new-york 样式。
- HSL 颜色现已转换为 OKLCH。

更多内容请阅读 [文档](/docs/tailwind-v4)。
