我们为 shadcn/ui 添加了新组件，并对 CLI 进行了大量改进。

以下是新内容的快速概览：

- [**轮播图**](#carousel) - 一个支持动画、滑动手势和键盘操作的轮播组件。
- [**抽屉**](#drawer) - 一个在移动端表现极佳的抽屉组件。
- [**分页**](#pagination) - 带有页面导航、上一页和下一页按钮的分页组件。
- [**可调整大小**](#resizable) - 用于构建可调整大小面板组和布局的组件。
- [**Sonner**](#sonner) - 你最后需要的吐司通知组件。
- [**CLI 更新**](#cli-updates) - 支持自定义 **Tailwind 前缀** 和 `tailwind.config.ts`。

### 轮播图

我们添加了一个功能完善的轮播图组件，支持动画、滑动手势和键盘操作。基于 [Embla Carousel](https://www.embla-carousel.com) 构建。

它支持无限循环、自动播放、垂直方向等功能。

### CLI 更新

这是最受欢迎的功能之一。现在你可以配置自定义的 Tailwind 前缀，CLI 会在添加组件时自动为你的实用类添加前缀。

这意味着你现在可以轻松将 shadcn/ui 组件添加到已有项目中，如 Docusaurus、Nextra 等。无冲突地作为你现有设计系统的插件。

```tsx /tw-/
<AlertDialog className="tw-grid tw-gap-4 tw-border tw-bg-background tw-shadow-lg" />
```

它兼容 `cn`、`cva` 以及 CSS 变量。

CLI 现在还能检测到 `tailwind.config.ts` 并自动为你添加 TypeScript 版本的配置。
