我们创建了一个开源注册表索引，您可以从中安装项目。

您可以搜索、查看和添加注册表索引中的项目，无需配置 `.components.json` 文件。

这些项目将自动添加到您的 `components.json` 文件中。

```bash
npx shadcn add @ai-elements/prompt-input
```

完整的注册表列表可访问 [https://ui.shadcn.com/r/registries.json](https://ui.shadcn.com/r/registries.json)。

要将注册表添加到索引中，请向 `shadcn/ui` 仓库提交 PR。更多详情请参见 [注册表索引文档](/docs/registry/registry-index)。
