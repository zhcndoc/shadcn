**今天，我们发布了 [**Blocks**](/blocks)**。

Blocks 是您可以用来构建应用的现成组件。它们完全响应式、无障碍且可组合，意味着它们是使用与 shadcn/ui 其他组件相同的原则构建的。

我们从仪表盘布局和身份验证页面开始，计划在未来几周添加更多 Blocks。

### 开源

Blocks 是开源的。您可以在 GitHub 上找到源码。在项目中使用它们，自定义它们并做出贡献。

### 请求一个 Block

我们还推出了“请求一个 Block”功能。如果您希望看到特定的 Block，只需在 GitHub 上创建一个请求，社区可以为其投票和构建。
