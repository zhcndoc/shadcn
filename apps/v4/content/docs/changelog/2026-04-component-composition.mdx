我们在组件文档中添加了**组合**部分，让您一目了然地看到正确结构：什么包裹什么，哪些子组件属于一起，以及如何避免无效嵌套。

```text
Card
├── CardHeader
│   ├── CardTitle
│   ├── CardDescription
│   └── CardAction
├── CardContent
└── CardFooter
```

## 我们为何添加此功能

我们发现，当**大语言模型和编码代理**能够看到完整结构时，它们组合元素会更可靠：更少的缺失包装器、更少的错误层次结构，以及与示例更好的匹配。

### 将文档带入您的代理

您或您的大语言模型可以从CLI中提取相同的组件文档（包括组合、用法和示例）到上下文中：

```bash
npx shadcn@latest docs card
```

如果您正在使用 [shadcn/skills](/docs/skills)，这将自动为您完成。
