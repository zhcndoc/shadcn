你可以使用 `shadcn` CLI 来运行你自己的代码注册中心。运行你自己的注册中心可以让你将定制的组件、hooks、页面、配置、规则及其他文件分发到任何项目中。

<Callout>
  **注意：** 该注册中心支持任何项目类型和任何框架，
  不仅限于 React。
</Callout>
