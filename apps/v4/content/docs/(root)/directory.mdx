这些注册表内置于 CLI，无需额外配置。要添加组件，请运行：`npx shadcn add @<registry>/<component>`。

<DirectoryList />

没看到想要的注册表？了解如何[在此添加](/docs/registry/registry-index)。

## 文档

你可以使用 `shadcn` CLI 来运行自己的代码注册表。运行你自己的注册表可以让你将自定义组件、hooks、页面、配置、规则及其他文件分发到任何项目。
