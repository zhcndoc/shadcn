你可以选择使用 CSS 变量（推荐）或工具类来进行主题定制。

## CSS 变量

```tsx /bg-background/ /text-foreground/
<div className="bg-background text-foreground" />
```

如果想使用 CSS 变量进行主题定制，请在你的 `components.json` 文件中将 `tailwind.cssVariables` 设置为 `true`。

```json {8} title="components.json" showLineNumbers
{
  "style": "new-york",
  "rsc": true,
  "tailwind": {
    "config": "",
    "css": "app/globals.css",
    "baseColor": "neutral",
    "cssVariables": true
  },
  "aliases": {
    "components": "@/components",
    "utils": "@/lib/utils",
    "ui": "@/components/ui",
    "lib": "@/lib",
    "hooks": "@/hooks"
  },
  "iconLibrary": "lucide"
}
```

## 工具类

```tsx /bg-zinc-950/ /text-zinc-50/ /dark:bg-white/ /dark:text-zinc-950/
<div className="bg-zinc-950 dark:bg-white" />
```

如果想使用工具类进行主题定制，请在你的 `components.json` 文件中将 `tailwind.cssVariables` 设置为 `false`。

```json {8} title="components.json" showLineNumbers
{
  "style": "new-york",
  "rsc": true,
  "tailwind": {
    "config": "",
    "css": "app/globals.css",
    "baseColor": "neutral",
    "cssVariables": false
  },
  "iconLibrary": "lucide",
  "aliases": {
    "components": "@/components",
    "utils": "@/lib/utils",
    "ui": "@/components/ui",
    "lib": "@/lib",
    "hooks": "@/hooks"
  }
}
```

## 命名规范

我们采用简单的 `background` 和 `foreground` 颜色命名规范。`background` 变量用于组件的背景色，而 `foreground` 变量用于文本颜色。

<Callout className="mt-4">

当变量用于组件背景色时，`background` 后缀会被省略。

</Callout>

假设有如下 CSS 变量：

```css
--primary: oklch(0.205 0 0);
--primary-foreground: oklch(0.985 0 0);
```

那么下面组件的背景色将是 `var(--primary)`，文本颜色为 `var(--primary-foreground)`。

```tsx
<div className="bg-primary text-primary-foreground">Hello</div>
```

## 添加新颜色

要添加新颜色，需要将它们添加到 CSS 文件中的 `:root` 和 `.dark` 伪类中。然后，使用 `@theme inline` 指令将颜色作为 CSS 变量提供使用。

```css title="app/globals.css" showLineNumbers
:root {
  --warning: oklch(0.84 0.16 84);
  --warning-foreground: oklch(0.28 0.07 46);
}

.dark {
  --warning: oklch(0.41 0.11 46);
  --warning-foreground: oklch(0.99 0.02 95);
}

@theme inline {
  --color-warning: var(--warning);
  --color-warning-foreground: var(--warning-foreground);
}
```

现在，你可以在组件中使用 `warning` 工具类。

```tsx /bg-warning/ /text-warning-foreground/
<div className="bg-warning text-warning-foreground" />
```

## 其他颜色格式

更多关于在 Tailwind CSS 中使用颜色的信息，请参阅 [Tailwind CSS 文档](https://tailwindcss.zhcndoc.com/docs/colors)。

## 基础颜色

可用的基础颜色包括：**中性色 (Neutral)**、**Stone**、**Zinc**、**Mauve**、**Olive**、**Mist** 和 **Taupe**。

你可以在初始化项目时通过预设或更新 `components.json` 文件中的 `baseColor` 字段来设置基础颜色。
