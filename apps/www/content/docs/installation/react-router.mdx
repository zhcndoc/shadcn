---
title: React Router
description: 安装并配置 shadcn/ui 以用于 React Router。
---

<Steps>

### 创建项目

```bash
npx create-react-router@latest my-app
```

### 运行 CLI

运行 `shadcn-ui` 初始化命令以设置你的项目：

```bash
npx shadcn@latest init
```

### 添加组件

现在你可以开始向你的项目添加组件。

```bash
npx shadcn@latest add button
```

上面的命令将把 `Button` 组件添加到你的项目。你可以像这样导入它：

```tsx showLineNumbers title="app/routes/home.tsx"
import { Button } from "~/components/ui/button"

import type { Route } from "./+types/home"

export function meta({}: Route.MetaArgs) {
  return [
    { title: "新的 React Router 应用" },
    { name: "description", content: "欢迎使用 React Router！" },
  ]
}

export default function Home() {
  return (
    <div className="flex flex-col items-center justify-center min-h-svh">
      <Button>点击我</Button>
    </div>
  )
}
```

</Steps>
