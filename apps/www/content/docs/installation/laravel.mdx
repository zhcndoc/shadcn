  **注意：** 以下指南适用于 Tailwind v4。如果您使用的是 Tailwind v3，请使用 `shadcn@2.3.0`。
</Callout>

<Steps>

### 创建项目

首先使用 Laravel 安装程序创建一个新的 Laravel 项目，包含 Inertia 和 React，命令为 `laravel new my-app`：

```bash
laravel new my-app --react
```

### 添加组件

现在您可以开始向项目中添加组件。

```bash
npx shadcn@latest add switch
```

上面的命令将会把 `Switch` 组件添加到 `resources/js/components/ui/switch.tsx`。然后您可以这样导入它：

```tsx {1,6} showLineNumbers
import { Switch } from "@/components/ui/switch"

const MyPage = () => {
  return (
    <div>
      <Switch />
    </div>
  )
}

export default MyPage
```

</Steps>
