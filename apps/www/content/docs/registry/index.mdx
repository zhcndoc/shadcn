您可以使用 `shadcn` CLI 来运行您自己的组件注册表。运行您自己的注册表可以让您将自定义组件、钩子、页面以及其他文件分发到任何 React 项目中。

注册表项目与 `shadcn` CLI 和 `Open in v0` 自动兼容。

## 要求

您可以根据自己的需要自由设计和托管自定义注册表。唯一的要求是您的注册表项目必须是有效的 JSON 文件，并符合 [registry-item schema specification](/docs/registry/registry-item-json)。

如果您想查看注册表的示例，我们有一个[模板项目](https://github.com/shadcn-ui/registry-template)可供您作为起点使用。
