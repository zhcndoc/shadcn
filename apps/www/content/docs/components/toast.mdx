## 使用方法

`useToast` 钩子返回一个 `toast` 函数，您可以用它来显示 toast。

```tsx
import { useToast } from "@/hooks/use-toast"
```

```tsx {2,7-10}
export const ToastDemo = () => {
  const { toast } = useToast()

  return (
    <Button
      onClick={() => {
        toast({
          title: "计划：跟进",
          description: "2023年2月10日星期五下午5:57",
        })
      }}
    >
      显示 Toast
    </Button>
  )
}
```

<Callout>

要同时显示多个 toast，您可以在 `use-toast.tsx` 中更新 `TOAST_LIMIT`。

</Callout>
