## 用法

```tsx
import {
  Table,
  TableBody,
  TableCaption,
  TableCell,
  TableHead,
  TableHeader,
  TableRow,
} from "@/components/ui/table"
```

```tsx
<Table>
  <TableCaption>你最近的发票清单。</TableCaption>
  <TableHeader>
    <TableRow>
      <TableHead className="w-[100px]">发票</TableHead>
      <TableHead>状态</TableHead>
      <TableHead>方式</TableHead>
      <TableHead className="text-right">金额</TableHead>
    </TableRow>
  </TableHeader>
  <TableBody>
    <TableRow>
      <TableCell className="font-medium">INV001</TableCell>
      <TableCell>已支付</TableCell>
      <TableCell>信用卡</TableCell>
      <TableCell className="text-right">$250.00</TableCell>
    </TableRow>
  </TableBody>
</Table>
```

## 数据表格

你可以使用 `<Table />` 组件构建更复杂的数据表格。将它与 [@tanstack/react-table](https://tanstack.com/table/v8) 结合，以创建具有排序、过滤和分页功能的表格。

有关更多信息，请查看 [数据表格](/docs/components/data-table) 文档。

你还可以在 [任务](/examples/tasks) 演示中查看数据表格的示例。
