## 用法

```tsx
import { Calendar } from "@/components/ui/calendar"
```

```tsx
const [date, setDate] = React.useState<Date | undefined>(new Date())

return (
  <Calendar
    mode="single"
    selected={date}
    onSelect={setDate}
    className="rounded-md border"
  />
)
```

有关更多信息，请参见 [React DayPicker](https://react-day-picker.js.org) 文档。

## 日期选择器

您可以使用 `<Calendar>` 组件构建日期选择器。有关更多信息，请参见 [日期选择器](/docs/components/date-picker) 页面。
