## 用法

```tsx
import {
  NavigationMenu,
  NavigationMenuContent,
  NavigationMenuIndicator,
  NavigationMenuItem,
  NavigationMenuLink,
  NavigationMenuList,
  NavigationMenuTrigger,
  NavigationMenuViewport,
} from "@/components/ui/navigation-menu"
```

```tsx
<NavigationMenu>
  <NavigationMenuList>
    <NavigationMenuItem>
      <NavigationMenuTrigger>选项一</NavigationMenuTrigger>
      <NavigationMenuContent>
        <NavigationMenuLink>链接</NavigationMenuLink>
      </NavigationMenuContent>
    </NavigationMenuItem>
  </NavigationMenuList>
</NavigationMenu>
```

## 示例

### 链接组件

当使用 Next.js 的 `<Link />` 组件时，可以使用 `navigationMenuTriggerStyle()` 来为触发器应用正确的样式。

```tsx
import { navigationMenuTriggerStyle } from "@/components/ui/navigation-menu"
```

```tsx {3-5}
<NavigationMenuItem>
  <Link href="/docs" legacyBehavior passHref>
    <NavigationMenuLink className={navigationMenuTriggerStyle()}>
      文档
    </NavigationMenuLink>
  </Link>
</NavigationMenuItem>
```

另请参见 [Radix UI 文档](https://www.radix-ui.com/docs/primitives/components/navigation-menu#with-client-side-routing)，以处理客户端路由。
