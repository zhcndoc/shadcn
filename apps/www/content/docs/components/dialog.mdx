## 用法

```tsx
import {
  Dialog,
  DialogContent,
  DialogDescription,
  DialogHeader,
  DialogTitle,
  DialogTrigger,
} from "@/components/ui/dialog"
```

```tsx
<Dialog>
  <DialogTrigger>打开</DialogTrigger>
  <DialogContent>
    <DialogHeader>
      <DialogTitle>您确定吗？</DialogTitle>
      <DialogDescription>
        此操作无法撤消。这将永久删除您的帐户并从我们的服务器中移除您的数据。
      </DialogDescription>
    </DialogHeader>
  </DialogContent>
</Dialog>
```

## 注意事项

要在 `上下文菜单` 或 `下拉菜单` 中激活 `Dialog` 组件，您必须将 `上下文菜单` 或 `下拉菜单` 组件包裹在 `Dialog` 组件中。有关更多信息，请参阅[此处](https://github.com/radix-ui/primitives/issues/1836)的相关问题。

```tsx {14-25}
<Dialog>
  <ContextMenu>
    <ContextMenuTrigger>右击</ContextMenuTrigger>
    <ContextMenuContent>
      <ContextMenuItem>打开</ContextMenuItem>
      <ContextMenuItem>下载</ContextMenuItem>
      <DialogTrigger asChild>
        <ContextMenuItem>
          <span>删除</span>
        </ContextMenuItem>
      </DialogTrigger>
    </ContextMenuContent>
  </ContextMenu>
  <DialogContent>
    <DialogHeader>
      <DialogTitle>您确定吗？</DialogTitle>
      <DialogDescription>
        此操作无法撤消。您确定要从我们的服务器中永久删除此文件吗？
      </DialogDescription>
    </DialogHeader>
    <DialogFooter>
      <Button type="submit">确认</Button>
    </DialogFooter>
  </DialogContent>
</Dialog>
```
